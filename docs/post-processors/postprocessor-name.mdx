# podman

Type: `podman`

<!--
  Include a short description about the post-processor. This is a good place

The podman post-processor is used to export Packer podman builds.


<!-- Post-Processor Configuration Fields -->

### Required

- `mock` (string) - The output path where to save exported build to.

<!--
  Optional Configuration Fields

### Optional

### Example Usage


```hcl
 source "podman" "example" {
   mock = "jay"
 }

 build {
   sources = ["source.podman.example"]

   post-processor "podman" {
     mock = "builds/podman.box"
   }
 }
```
